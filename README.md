## 🏗️ Architecture I swear by

```
Controller → Binding → Screen   (GetX, always)
Hive                            (local storage, offline-first)
Paise-based storage             (no float bugs on money)
RLS via Supabase                (row-level security for multi-tenant)
WebSockets                      (real-time booking & tracking)
```
